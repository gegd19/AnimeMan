👋
欢迎使用 AnimeMan · 番剧智能管家！

检测到您尚未完成基本配置，请跟随教程快速上手。

1️⃣

获取 TMDB API Key

前往 TMDB 官网 注册账号并申请 API 密钥（免费）。

2️⃣

填写基础配置

在下方配置面板中填入 API Key、源文件夹和目标文件夹（剧集/电影）。

🤖

启用 AI 智能解析

在「AI 解析设置」中勾选启用 AI 解析，并填入 AI API Key。
⚡ 建议开启「文件夹批量解析」以节省 Token 费用。
⚠️ 若未填写 Key，系统将自动禁用并回退到正则解析。

3️⃣

保存并测试连通性

点击「💾 保存配置」，然后点击「🔌 测试连通性」确保 TMDB 网络正常。

4️⃣

开始处理任务

点击顶部「▶ 开始处理」，系统将自动扫描并整理媒体文件。AI 会在文件名不规范时辅助识别。

💡 AI 功能可选，但强烈推荐开启以提升识别准确率。部分服务商提供免费额度，请自行了解。
