## [1.0.0] - 2026-04-19

### 🎉 Initial Release - AnimeMan 正式发布

经过全面重构与优化，**AnimeMan · 番剧智能管家** 首个正式版本发布。
专为 NAS 用户与动漫收藏家设计的全自动媒体库整理工具。

### ✨ Added - 新增功能

#### 核心解析增强
- **罗马数字季号自动转换**：支持识别文件名中的罗马数字季号（如 `III`、`IV`、`X`）并自动转换为 `Sxx` 格式，配合 anitopy 精准解析。
- **总集数跨季自动换算**：根据 TMDB 季集结构，自动将绝对集号（如第 1072 集）换算为正确的季号和季内集号。
- **特辑/OVA 手动映射规则**：支持用户自定义关键词规则，强制指定 TMDB ID 和季/集号，完美处理 OVA、SP、剧场版等特殊内容。
- **文件夹级 AI 批量解析**：一次 API 调用解析整个文件夹的视频文件，大幅降低 AI Token 消耗，提升处理速度。

#### 媒体库管理面板
- **电影与剧集彻底分离**：支持「全部 / 仅电影 / 仅剧集」视图切换，互不干扰。
- **独立无限滚动加载**：电影和剧集网格各自实现分页加载，滚动到底部自动追加，轻松承载数万级媒体库。
- **集数详情懒加载**：仅当用户首次展开某季时才渲染集数列表，初始渲染 DOM 节点减少 90%+，页面响应飞速。
- **批量修正与删除**：支持勾选多个电影/单集/整部剧集，一键批量重新匹配或删除目标文件。
- **单集源文件详情浮层**：点击集数标题可查看完整源文件路径、处理时间等信息。

#### 字幕智能中心
- **AI 分组智能匹配**：同一剧集的字幕文件自动分组，调用 AI 解析标题并批量匹配到正确集数。
- **强制匹配模式**：支持手动选择目标电影或指定剧集季号，强制对应勾选的字幕文件。
- **本地字幕扫描与自动匹配**：基于模糊匹配算法，自动关联本地字幕与媒体库条目。
- **ffsubsync 自动调轴**：支持调用 ffsubsync 同步字幕时间轴（需自行安装）。

#### 离线 AI 预解析
- 提供「扫描源文件夹」与「全量预解析」功能，可提前生成 AI 解析缓存，实时处理时直接调用，降低延迟与费用。

#### Web 界面体验
- **移动端完整适配**：双列卡片网格、表格卡片化布局、触摸区域 ≥ 44px、模态框全屏优化、输入框垂直排列。
- **桌面端视觉增强**：卡片玻璃态设计、细腻阴影、悬浮动画、表格行悬停效果。
- **品牌全面升级**：项目更名为 **AnimeMan**，添加 ASCII 艺术启动横幅与 Web 标题。
- **引导教程横幅**：首次使用显示配置引导，降低上手门槛。

#### 部署与运维
- **Docker 支持**：提供 `Dockerfile` 与 `docker-compose.yml`，支持 x86/ARM 架构 NAS 一键部署。
- **错误日志自动轮转**：使用 `RotatingFileHandler`，单文件最大 10MB，保留 1 个备份，防止日志爆炸。
- **优雅处理中断信号**：命令行模式捕获 `Ctrl+C`，安全保存缓存后退出，避免线程警告。
- **端口统一为 8000**：避免与其他常见服务冲突。

### 🚀 Performance Improvements - 性能优化

- **媒体库前端渲染重构**：从全量渲染改为分页 + 虚拟滚动 + 懒加载，10 万文件场景下内存占用降低 80%，首屏渲染时间缩短至 1 秒内。
- **TMDB 搜索结果缓存**：增加 7 天 TTL 缓存，减少重复搜索请求。
- **文件夹 AI 解析缓存**：基于文件指纹的持久化缓存，避免同一文件夹重复调用 AI。
- **增量处理默认开启**：仅扫描新增或指纹变化的文件，大幅缩短日常扫描时间。

### 🐛 Bug Fixes - 问题修复

- 修复移动端 TMDB API Key 输入框过小、文字不显示的问题。
- 修复批量修正模态框中 TMDB 搜索框布局错乱、输入文字不显示的问题。
- 修复已处理缓存管理界面移动端信息过多、横向滑动困难的问题（改为卡片化布局）。
- 修复失败缓存列表移动端操作按钮过小、分布零散的问题。
- 修复 `main.py` 中 `--test-tmdb` 参数因缩进错误导致的语法问题。
- 修复 `Ctrl+C` 中断时线程异常警告，改为优雅提示。

### 🔧 Changed - 调整与优化

- 将 `fuzzywuzzy` 依赖替换为活跃维护的 `thefuzz`，修复潜在的安全与兼容性问题。
- 升级 `requests` 至 `>=2.32.0`，修复 CVE-2024-35195 漏洞。
- 优化 `auto_config.json` 结构，增加 `ignore_patterns` 默认值（如 `SPs`、`PV`），自动跳过特典目录。
- 批量修正文件列表移除冗余路径前缀，界面更清爽。
